# База данных музыкальных альбомов

Консольное приложение для создания, хранения и управления информацией о музыкальных альбомах.

---

##  Назначение

Проект демонстрирует практическое применение структур данных, файлового ввода-вывода и алгоритмов поиска/сортировки в языке C. Может использоваться как учебный пример или основа для персональной музыкальной коллекции.

---

##  Функциональные возможности

### Основной функционал:
- CRUD-операции: Создание, чтение, обновление записей альбомов
- Поиск: По названию альбома
- Сортировка: По году выпуска
- Сохранение: Бинарный формат для долговременного хранения
- Консольный интерфейс: Интерактивное меню с 8 командами
- Фильтрация: Автоматический вывод альбомов с количеством треков ≥ 10 и длительностью ≤ 50 мин

### Структура данных альбома:

| Поле | Тип | Описание |
|------|-----|----------|
| title | char[100] | Название альбома |
| releaseYear | int | Год выпуска |
| style | char[50] | Музыкальный стиль |
| trackCount | int | Количество треков |
| durationMinutes | int | Продолжительность в минутах |

##  Сборка и запуск

### Требования:
- Компилятор C (GCC/Clang/MSVC)
- Стандарт C99 или новее
- Windows/Linux/macOS

### Сборка:
# Вручную
gcc -std=c99 -o music_albums main.c database.c fileio.c utils.c

# Или используйте compile.bat (Windows)
compile.bat
Запуск:

./music_albums      # Linux/macOS
music_albums.exe    # Windows

## Как использовать:

1. **Создайте папку проекта** `music-albums-db`
2. **Создайте все файлы** из списка выше с соответствующим содержимым
3. **Для сборки на Windows**: просто запустите `compile.bat`
4. **Для сборки на Linux/macOS**:
   
bash
   gcc -std=c99 -o music_albums main.c database.c fileio.c utils.c
   ./music_albums
`

Ваш исходный код полностью интегрирован в систему:

· Структура MusicAlbum - это ваши данные об альбомах
· Функция print_album() - ваша функция вывода
· Тестовые данные - ваши 5 альбомов
· Автоматическая фильтрация по условию (≥10 треков, ≤50 минут) работает при каждом выводе всех альбомов
